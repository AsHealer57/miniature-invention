About FurtherLearning

FurtherLearning is a beginner-friendly platform created to help students and aspiring professionals start their journey in cloud computing and AWS (Amazon Web Services).

Whether you're learning with a mobile phone or on a PC, our aim is to make cloud education accessible, practical, and understandable for everyone. You’ll find basic lessons, real-world project examples, and support for working on cloud technologies step-by-step.

This platform is proudly built by Fizza Abdul Razzaq, a BSCS student passionate about tech education, accessibility, and making cloud learning possible for everyone — even without paid tools.
